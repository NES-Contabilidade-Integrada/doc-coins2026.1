# Convenções do Projeto

## Títulos de documentos

Não incluir "COIN'S" ou "COINS" no título (`# Heading`) de nenhum documento. O repositório já pertence ao projeto, tornando o nome redundante.

**Correto:** `# Dicionário de Dados`  
**Errado:** `# Dicionário de Dados – Sistema COIN'S`

A exceção é o `docs/index.md`, que é a homepage e apresenta o projeto pelo nome completo.
