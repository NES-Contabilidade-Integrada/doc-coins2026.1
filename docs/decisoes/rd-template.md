# Template de RD

Use o modelo abaixo ao registrar uma nova decisão no [Diário de Decisões](decisoes.md).

---

```markdown
## RD XXX — Título da Decisão

- **Data:** AAAA-MM-DD
- **Decisores:** [nomes ou papéis]

#### Contexto

Descreva o problema ou necessidade que motivou a decisão.

#### Decisão

Descreva o que foi decidido.

#### Consequências

Liste os impactos positivos e negativos esperados da decisão.
```
